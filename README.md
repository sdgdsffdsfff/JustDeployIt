JustDeployIt
============

用于完成基于脚本语言的WEB应用程序的部署



## 规则 ##

### 邮件发送规则 ###

1. 项目创建人将会收到所有项目配置更改的变动通知
2. 项目创建人将会收到所有部署操作结果的通知
3. 部署创建人将会收到部署结果的通知

### 页面跳转规则 ###

1. 项目工程创建过程中，按 5 步骤顺序前进
2. 其它页面进入到配置页，完成工作后跳回入口页面

### 部署规则 ###

1. 默认情况下只覆盖，不删除
2. 删除目标服务器的文件，只发生在明确执行远端删除的情况下

## 实体关系 ##

1. 配置文件跟服务器组直接关联
2. 排除文件跟项目直接关联
